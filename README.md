# test-assignment-frontend

##Тестовое задание

### О проекте:

Стек используемых технологий:
* React.js
* Redux, React-Redux
* TypeScript
* Axios
* Webpack

### Основные команды для работы:

`npm install` - установка зависимостей

`npm start` - запуск локального сервера

`npm run build` - сборка проекта
